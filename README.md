# Microsite Starter

Microsite is a next-generation static site generator.

## Getting Started

To start a new project using Microsite, clone this repo to a new directory:

```
npm init microsite
```

To spin up a development server, run:

```
npm start
```

To build your production-ready site, run:

```
npm run build
# or build + serve with
npm run serve
```
